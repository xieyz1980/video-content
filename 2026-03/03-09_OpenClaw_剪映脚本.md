# 剪映视频剪辑脚本

**视频主题：** 第一次用OpenClaw，千万别本地部署！
**制作工具：** 剪映（CapCut）
**成片时长：** 3分钟
**建议：** 直接用剪映【图文成片】+ 智能配音，最快15分钟出片

---

## 🎬 剪映时间轴脚本

### 【第一段】开场抓人（0:00-0:15）

| 属性 | 内容 |
|------|------|
| **画面** | 真人出镜 / 或用"数字人"功能 |
| **配音** | 剪映智能配音 - 选择"新闻男声"或"解说男声" |
| **字幕** | 动态字幕，逐字出现 |
| **背景音乐** | 紧张感BGM（可选"商业科技"分类） |
| **特效** | "千万别本地部署"红底白字放大弹出 |

**文案（配音用）：**
> 想试OpenClaw的老板，先听我说完！第一次用，千万别本地部署——clouds only！今天告诉你3个原因，帮你省几千块冤枉钱。

**剪映操作：**
1. 点击【图文成片】→ 粘贴文案
2. 选择【智能配音】→ "新闻男声"
3. 点击【生成视频】
4. 替换生成的画面为你自己拍摄的画面（可选）

---

### 【第二段】原因一：找场景（0:15-0:50）

| 属性 | 内容 |
|------|------|
| **画面** | 剪映【文字视频】功能 |
| **配音** | 同上 |
| **字幕** | 大号文字，关键词高亮 |
| **贴纸** | 添加"💡" "💰"图标 |
| **转场** | 从左滑入 |

**文案：**
> 原因一：你还没找到使用场景。OpenClaw能干很多事，自动回邮件、写周报、管日程、监控数据。但哪些对你有用？你不试试怎么知道？云部署5分钟搞定，零成本试错。本地部署先花2万买设备，结果发现用不上——这2万谁买单？

**剪映操作：**
1. 新建【文字视频】
2. 输入文案，选择"商务"模板
3. 设置文字动画：逐行出现
4. 添加贴纸：💡在"OpenClaw"旁，💰在"2万"旁

---

### 【第三段】原因二：风险高（0:50-1:25）

| 属性 | 内容 |
|------|------|
| **画面** | 左右分屏对比（剪映【画中画】） |
| **左侧** | 文字：云部署 = 拎包入住 ✅ |
| **右侧** | 文字：本地部署 = 自己盖房 ❌ |
| **配音** | 同上 |
| **音效** | 添加"警示音" |

**文案：**
> 原因二：本地部署风险高。网络穿透会配吗？防火墙怎么开？出问题了谁解决？云部署是拎包入住，本地部署是自己盖房。你连OpenClaw是啥都没搞懂，就开始搞服务器？步子太大了！

**剪映操作：**
1. 添加【画中画】轨道
2. 左侧视频/文字：云部署优势
3. 右侧视频/文字：本地部署难点
4. 添加音效：警示音（在"风险高"处）

---

### 【第四段】原因三：算总账（1:25-2:00）

| 属性 | 内容 |
|------|------|
| **画面** | 剪映【表格】功能或图文卡片 |
| **配音** | 同上 |
| **动画** | 数字滚动效果（成本对比） |
| **贴纸** | "健身房"图标 + 💸 |

**文案：**
> 原因三：成本要算总账。有人说本地长期更省钱——前提是你要长期用！第一次试用，能不能坚持3个月都是未知数。这时候谈省钱，就像办健身房年卡，去了3次就闲置。云部署一个月几十块，用3个月才几百块。摸清了价值，再决定要不要本地——这才叫理性！

**剪映操作：**
1. 添加【图文卡片】模板
2. 设置对比表格：
   - 云部署：月费50-150元
   - 本地部署： upfront 2万
3. 在"健身房"处添加相关贴纸

---

### 【第五段】正确姿势流程（2:00-2:30）

| 属性 | 内容 |
|------|------|
| **画面** | 剪映【流程图】模板或箭头动画 |
| **配音** | 同上 |
| **动画** | 步骤依次点亮 |
| **背景音乐** | 转为轻快积极风格 |

**文案：**
> 听好了，标准流程：第一步云部署，5分钟跑起来；第二步用2到4周找场景；第三步确定长期用，才考虑本地部署。记住：先云、找场景、再本地。

**剪映操作：**
1. 搜索【流程图】模板
2. 设置三步：
   - Step1: ☁️ 云部署（5分钟）
   - Step2: 🔍 找场景（2-4周）  
   - Step3: 🖥️ 本地部署（确定长期用）
3. 添加箭头动画连接三步

---

### 【第六段】结尾号召（2:30-3:00）

| 属性 | 内容 |
|------|------|
| **画面** | 真人出镜 / 金色文字特效 |
| **配音** | 语气坚定，节奏放缓 |
| **字幕** | 金句大字停留3秒 |
| **贴纸** | 👍 点赞图标 + 💬 评论图标 |
| **背景音乐** | 收尾，渐弱 |

**文案：**
> 最后总结一句话：云部署是试错的成本，本地部署是确定价值后的选择。第一次用OpenClaw的老板，听我的，先上云！评论区告诉我你准备怎么部署？下期教你怎么5分钟云部署跑起来，关注别走丢！

**剪映操作：**
1. 添加【文字特效】→ 金色发光字体
2. 输入金句："云部署是试错的成本，本地部署是确定价值后的选择"
3. 设置停留时间：3秒
4. 添加【关注】贴纸动画

---

## 🔧 剪映快速出片流程（15分钟）

### 方法一：图文成片（最快）

1. 打开剪映 → 点击【图文成片】
2. 粘贴完整文案：
   > （复制上方所有文案拼接）
3. 选择【智能配音】→ "新闻男声"
4. 点击【生成视频】
5. 替换不满意画面，调整字幕样式
6. 导出

### 方法二：手动剪辑（更精细）

1. 导入素材（真人出镜视频/图片）
2. 按上方时间轴分段添加
3. 每段添加字幕、配音、特效
4. 调整转场、背景音乐
5. 导出
